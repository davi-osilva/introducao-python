# Exercícios — Introdução à Programação com Python

Resolução de exercícios do livro **Introdução à Programação com Python**, de Nilo Ney Coutinho Menezes.

## Objetivo

Este repositório foi criado com o objetivo de praticar lógica de programação e reforçar os conceitos apresentados no livro, utilizando a linguagem Python.

## Conteúdo

Os exercícios abordam temas como:

- Variáveis e tipos de dados
- Condicionais
- Estruturas de repetição
- Listas
- Funções
- Strings
- Dicionários
- Arquivos
- Programação orientada a objetos

## Tecnologias

- Python 3

## Organização

Os exercícios estão separados por capítulos para facilitar a navegação e o acompanhamento do progresso.

## Observação

Este repositório possui fins exclusivamente educacionais e de prática pessoal.
